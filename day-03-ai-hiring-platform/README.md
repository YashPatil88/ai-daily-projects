# AI Hiring & Interview Intelligence Platform

## Overview
A SaaS platform for automated AI-driven candidate interviews and recruiter evaluation.

## Features
- Recruiter & Candidate roles
- AI-generated interview questions
- AI-based answer scoring
- Recruiter dashboard

## Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Frontend Setup
```bash
cd frontend
npm install
npm start
```
